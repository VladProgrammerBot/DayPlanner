# Day Planner

Простий CLI-додаток, що будує детальний план дня на основі:
- діапазону планування;
- фіксованих активностей (з чітким часом);
- гнучких активностей (без фіксованого часу, але з пріоритетом);
- резерву "спонтанного" часу.

## Запуск

```bash
python3 planner.py --input example_input.json
```

## Формат вхідних даних

```json
{
  "planning_range": {"start": "08:00", "end": "22:00"},
  "fixed_activities": [
    {"name": "Назва", "start": "11:00", "end": "12:00"}
  ],
  "flexible_activities": [
    {"name": "Назва", "duration_minutes": 60, "priority": 3}
  ],
  "spontaneous_minutes": 90
}
```

## Логіка планування

1. Додаються фіксовані активності.
2. Вільні вікна діляться та в них резервується `spontaneous_minutes` (пропорційно до довжини вікон).
3. Гнучкі задачі плануються у залишок вікон у порядку пріоритету (від більшого до меншого).
4. Якщо для задач не вистачило часу, це виводиться окремим списком.
